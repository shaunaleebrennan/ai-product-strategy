# The Prototype Bet

## What I Built
A working AI comms agent (Liquid Content) that turns one internal communications prompt into multi-channel content, each output generated specifically from the brief, with a predicted engagement score, optimal send time, and AI reasoning for each recommendation.
Key features built:

- File upload — attach a PDF, Word doc, deck or text file as source material alongside the prompt

- Real AI generation — every output is customised to the specific prompt, not templated

- Channel selection — three default channels (Article, Video, Podcast) generated on first run, with six additional channels available on demand: Workvivo TV, Infographic, Post, Billboard, Event, and Journey

- Publish control — each content card has a checkbox to select what gets published, with a live count in the header 

- Per-card regeneration — Regenerate re-calls the AI for that channel only, without regenerating everything

## The Bet in One Sentence
Internal communicators will adopt Workvivo's AI Comms Agent (Liquid Content) not because it writes better content, but because it is the only tool that tells them whether their content will actually land, before they hit publish.

## Kill Criteria
Microsoft Copilot or OpenAI ships a native multi-format comms tool with intranet integrations at no extra cost to enterprise customers within 12 months
Workvivo's engagement and sentiment data does not demonstrably improve content performance scores over time, the data flywheel fails to compound
Internal communicators continue using ChatGPT personally for drafting and treat Workvivo as distribution only, never shifting their creative workflow into the platform
